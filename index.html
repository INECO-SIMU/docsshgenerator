Usage

Usage: script.sh -e extension -f filepath -o outputdir -n numlimit -c cat -a airport -m yearmonth -r region -u upload -F filterRegion -C Checkcsv

Mandatory Options

  • -a Airport OACI code
  • -c Radar CAT
  • -m Year and month in format ym, example: 2207

Optional Options
           
  • -e File extension, in most cases it would be gps. By default this value is gps
    •        
  • -f Asterix files path. By default this value is C:/Users/ADMINI~1/AppData/Local/Temp.
    (Don't need to change the direction of the slashes)
    •        
  • -h Sows help menu
    •        
  • -l Only uses local files indicated with the -f param. By default this value is false
    •        
  • -o Output directory. By default this value is C:/Users/ADMINI~1/AppData/Local/Temp.
    (Don't need to change the direction of the slashes)
    •        
  • -n Limit the number of decodifications to a given number. By default this value is 6
    •        
  • -r Region/Airport to download the files. By default this take the value of the Airport param -a
    •        
  • -u If true, files will be stored in S3 an not in local storage. By default value is true
    •        
  • -F If true, files will be filtered by airport. By default value is true
    •        
  • -C If true, csv decoded files will be downloaded from S3 if exists before decoding. By default value is false"
